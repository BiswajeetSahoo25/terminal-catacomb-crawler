# Terminal Dungeon Crawler

A classic roguelike dungeon crawler game built for the terminal with ASCII graphics.

## Features (Planned)
- [ ] ASCII-based graphics
- [ ] Procedurally generated dungeons
- [ ] Turn-based combat
- [ ] Character progression
- [ ] Save/load system
- [ ] Multiple enemy types

## Installation
```bash
git clone https://github.com/DimitrovK/terminal-dungeon-crawler.git
cd terminal-dungeon-crawler
pip install -r requirements.txt
python main.py
```

Controls

WASD or Arrow keys: Move
Q: Quit
I: Inventory (coming soon)
